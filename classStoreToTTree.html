Detailed Description

StoreToTTree::MakeBranches takes in a Store and makes a TTree branch for each variable in the Store. StoreToTTree::FillBranches transfers the current contents of the BStore into a new TBranch entry. StoreToTTree accepts both (ASCII) Store class and (Binary) BStore classes.
