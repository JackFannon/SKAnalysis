Detailed Description

A simple tool to plot basic characteristics of neutron captures in SKG4, reading outputs from Harada-san's SecondaryInfo arrays. These represent largely the same variable set as from the atmpd information available via fillnt_simple and h2root. There are, however, minor differences accounted for in this tool compared to the TruthNeutronCaptures tool used for skdetsim->fillnt_simple->h2root files.
