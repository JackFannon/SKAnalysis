\doxysubsection{Detailed Description}
Store\+To\+TTree\+::\+Make\+Branches takes in a Store and makes a TTree branch for each variable in the Store. Store\+To\+TTree\+::\+Fill\+Branches transfers the current contents of the BStore into a new TBranch entry. \mbox{\hyperlink{classStoreToTTree}{Store\+To\+TTree}} accepts both (ASCII) Store class and (Binary) BStore classes.
