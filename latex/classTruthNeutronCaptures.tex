\doxysubsection{Detailed Description}
A simple tool to plot basic characteristics of neutron captures, reading outputs from skdetsim.
\begin{DoxyEnumerate}
\item run skdetsim or skdetsim-\/gd with output option \textquotesingle{}SKCNTL-\/\+OUTPUTTYPE 2\textquotesingle{} to generate zbs output file
\item convert the zbs file to an hbk file with {\ttfamily fillnt\+\_\+simple.\+sh -\/o (output hbook file) (input zbs file)}
\item convert the hbk file to a root file with {\ttfamily h2root (input hbook file) (output root file)} This somewhat convoluted process is required to retain information about neutrons and gammas. This file processes the output root files from step 3.
\end{DoxyEnumerate}
