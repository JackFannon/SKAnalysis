\doxysubsection{Detailed Description}
A simple tool to plot basic characteristics of neutron captures, reading outputs from skdetsim.
\begin{DoxyEnumerate}
\item checkout and compile modified SKOFL with secondaries added to MCInfo class.
\item compile modified skdetsim with NEUTRON environmental variable defined to enable saving secondaries.
\item run skdetsim with option \textquotesingle{}SKCNTL-\/\+OUTPUTTYPE 1\textquotesingle{} and \textquotesingle{}SKCNTL-\/\+FILEFORMAT 1\textquotesingle{} to generate SKROOT output. This file processes the output root files from step 3.
\end{DoxyEnumerate}
