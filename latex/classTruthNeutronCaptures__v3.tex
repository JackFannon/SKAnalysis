\doxysubsection{Detailed Description}
A simple tool to plot basic characteristics of neutron captures in SKG4, reading outputs from Harada-\/san\textquotesingle{}s Secondary\+Info arrays. These represent largely the same variable set as from the atmpd information available via fillnt\+\_\+simple and h2root. There are, however, minor differences accounted for in this tool compared to the \mbox{\hyperlink{classTruthNeutronCaptures}{Truth\+Neutron\+Captures}} tool used for skdetsim-\/$>$fillnt\+\_\+simple-\/$>$h2root files.
